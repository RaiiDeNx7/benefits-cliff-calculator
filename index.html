Same as Medicaid!N214 (L214+M214).

HCV program (current) —

Same as Housing Voucher!Q209 (monthly $). Shown when Housing Choice Voucher is selected. Uses locality, household size, adult ages, bedrooms, earned income, optional TANF and Social Security fields.

HCV program (new income) —

Same as Housing Voucher!Q211.

TANF (current) —

Same as MAX('TANF-VIEW'!L212,'TANF-VIEW'!T212) (monthly $). Shown when TANF is selected. Uses locality, earned income, and Social Security.

TANF (new income) —

Same as MAX('TANF-VIEW'!L214,'TANF-VIEW'!T214). Row 214 still uses current packaged monthly earned on the TANF B column (workbook B90), same as row 212.

TANF/VIEW (current) —

Same as 'TANF-VIEW'!T212 (monthly $). Shown when TANF/VIEW is selected.

TANF/VIEW (new income) —

Same as 'TANF-VIEW'!T214. Row 214 uses current packaged earned (B90) on the TANF B column, same as T212.

WIC (current) —

Same as WIC!F211 (monthly $). Shown when WIC is selected. Counts children in infant through preschool bands; income test includes earned income, Social Security, and MAX(TANF-VIEW L,T).

WIC (new income) —

Same as WIC!F213. Uses new monthly earned for B213.
